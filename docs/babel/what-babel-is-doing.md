# Babel이 하는 일과 하지 않는 일

1. Babel이 하는 일: ES6+로 작성한 최신 문법을 지원 대상 브라우저에서 호환 가능한 문법으로 변경한다.
2. Babel이 하지 않는 일: 지원 대상 브라우저에서 필요로 할 지도 모르는 새로운 속성을 전역 네임스페이스 또는 원시 자료형에 추가하지는 않는다.
